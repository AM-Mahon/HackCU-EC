    Lesson 2: variables
  1. “This is a variable. A variable is like a box that contains letters and numbers.”
  2. “The box has a label this is called a variable name”
  3. “You can label a box and fill it with letters and numbers. In programming this is called variable declaration.”
  4. “Variables have names, you can think of this as the writing on a box. For example, you may have a box labeled “storage” in your house.”
  5. Variables also have contents. You can think of this as the items of the box.
  6. The first part of this lesson will involve variable declaration. It will involve pushing a bottom labeled “create variable. From there it will instruct them to pick a color. Then it will tell them to label the variable box. Next it asks them to write the contents of the box.
  7. After declaring their first variable the program will tell them to create1 variable that contains a letter, one that contains a number, and one that creates multiple letters
  8. As they “declare” variables they will appear in a drag and drop menu
  9. Next, it will move onto part two. “Variables can be put in print statements. When this is done by contents of the variables will be printed.
  10. “Try printed all the variables you declared”
  11. “Try printing this statement “statement here” by putting it in a variable and printing it.
  12. “You can always take objects out of a box and put new objects in it. You can do the same with variables. Try changing the value of a variable”
